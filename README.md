# ms-converters

### Start module

    $ go NAME_PROJECT src/java/main
    $ start MAIN_CLASS
    
### Start project Local

    $ run cofig_server
    $ run service_registry
    $ run service_gateway
    $ run other modules

### Create JAR files 

    $ run file "run-package.bat"
    
### Create Docker images and start Docker containers 

    $ run file "run-install.bat"
    $ cd dir project
    $ docker-compose up -d
